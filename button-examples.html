CrafteriAuth Button Examples

This page provides ready-to-use button examples for integrating CrafteriAuth into your website.

Remember to replace https://your-website.com/crafteriauth/return with your actual callback URL.

Default Style Button

Login with Crafteri 
<a href="https://auth.crafteri.net/login?service=https://your-website.com/crafteriauth/return" class="crafteri-button-default">
    Login with Crafteri
</a>

<style>
.crafteri-button-default {
    display: inline-block;
    background-color: #4285f4;
    color: white;
    padding: 10px 16px;
    border-radius: 4px;
    font-weight: 500;
    text-decoration: none;
    font-size: 14px;
    transition: background-color 0.3s;
}

.crafteri-button-default:hover {
    background-color: #3367d6;
}
</style>
