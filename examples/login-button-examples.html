Adjusting Button Roundness

To make the buttons more or less rounded, modify the border-radius property in the .crafteri-btn class:

  • border-radius: 4px; - Less rounded (more square corners)
  • border-radius: 6px; - Moderately rounded (current setting)
  • border-radius: 8px; - More rounded
  • border-radius: 20px; - Very rounded
  • border-radius: 25px; - Pill-shaped

Adjusting Logo Size in Circular Buttons

To change the logo size in circular buttons:

  1. First, adjust the button container size by changing the width and height values in the .crafteri-btn-circle class
  2. Then, adjust the logo size by changing the max-height and max-width values in the .crafteri-btn-circle .crafteri-logo class

For example, to make the logo larger:

.crafteri-btn-circle {
    width: 96px;  /* Larger container */
    height: 96px; /* Larger container */
    /* other properties remain the same */
}

.crafteri-btn-circle .crafteri-logo {
    /* height and width remain auto */
    max-height: 72px; /* Larger logo */
    max-width: 72px;  /* Larger logo */
    /* other properties remain the same */
}

Important Notes

  • Replace ../logo.png with the path to your actual Crafteri logo image
  • For optimal results, use a transparent PNG logo with dimensions of at least 48x48px
  • For better performance, consider using Google Fonts to load Roboto: <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@500&display=swap" rel="stylesheet">
  • These buttons are designed to be responsive and will work well on mobile devices
  • The buttons include hover and active states for improved user interaction
