CrafteriAuth Login Button Examples

Here are several examples of how you can implement the "Login with Crafteri" button in your application. Remember to replace YOUR_REDIRECT_URL with your actual callback URL.

Standard Blue Button

Login with Crafteri

HTML Code:

<a href="https://auth.crafteri.net/login?service=YOUR_REDIRECT_URL" class="crafteri-btn crafteri-login-btn-blue">
    Login with Crafteri
</a>

<style>
    .crafteri-btn {
        display: inline-block;
        padding: 10px 15px;
        border-radius: 4px;
        text-decoration: none;
        font-weight: bold;
        font-size: 14px;
    }
    
    .crafteri-login-btn-blue {
        background-color: #1a73e8;
        color: white;
        border: none;
    }
</style>
